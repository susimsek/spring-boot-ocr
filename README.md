## Prerequisites

* Java 11
* Maven 3.3+
* Docker 19.03+
* Docker Compose 1.25+

## Installation


```sh
./mvnw compile jib:dockerBuild
```

```sh
docker-compose up -d 
```

## Used Technologies

* Spring Boot 2.4.3
* Tesseract  
* Opencv  
* Spring Boot Web 
* Problem Spring Web
* SpringDoc Openapi WebMvc Core
* SpringDoc Openapi Ui
* Tess4j
* Spring Boot Validation
* Spring Boot Actuator
* Maven Jib Plugin
* Maven Clean Plugin
* Maven Enforcer Plugin
* Lombok
* Dev Tools
* Spring Boot Test
